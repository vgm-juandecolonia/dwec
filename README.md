# Proyecto DWEC

### Apuntes

Inicialización de un projecto de Node.js

```powershell
npm init
```

Ejecución del script por consola

```powershell
node {script.js}
```

Instalacion de modulos:

```powershell
npm i {module_name}
```

Desinstalacion de modulos:

```powershell
npm uninstall {module_name}
```
